# Sci-Fi-Fps
A Sci-Fi fps game.
If you have any question, please, let me know.

## Engine
* Unity

## Features
* Configurable parameters for Enemies and player
* Camera animations
* Enemy finds and attack player

## Controls

* LMB - Shoot
* RMB - Aim
* W - Forward
* S - Backward
* A - Left
* D - Right
